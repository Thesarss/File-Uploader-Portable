## 📝 Informasi Aplikasi

### Nama Aplikasi
**File Uploader Portable**

### Deskripsi Singkat
Aplikasi web modern untuk upload file dengan kategorisasi otomatis dan akses online

### Spesifikasi Teknis

**Frontend:**
- React 18.3.1
- TypeScript 5.6.2
- Vite 6.0.1
- Tailwind CSS 3.4.17
- shadcn/ui components
- Axios untuk HTTP client

**Backend:**
- Node.js 18+
- Express 4.21.2
- TypeScript 5.7.3
- PostgreSQL 14+
- Multer untuk file upload
- Jest untuk testing

**Infrastructure:**
- Cloudflare Tunnel untuk online access
- Local file system storage
- Docker support untuk database

### Bahasa Pemrograman
- TypeScript: 95%
- JavaScript: 3%
- SQL: 2%

### Fitur Utama
1. Multiple file upload dengan drag & drop
2. Kategorisasi otomatis (Photo, Video, Document, Audio, Archive, Other)
3. Chunked upload untuk file besar (≥50MB)
4. Upload history dengan database
5. UI responsive (desktop & mobile)
6. Online access via Cloudflare Tunnel (Quick & Named)
7. Real-time progress tracking
8. CORS protection & rate limiting
9. File validation & compression

### Cara Setup
1. Clone repository
2. Install dependencies (npm install)
3. Setup database (PostgreSQL)
4. Configure .env files
5. Build aplikasi (npm run build)
6. Jalankan dengan script .bat

### Cara Penggunaan
- **Lokal:** `start-all.bat` → http://localhost:4173
- **Online (Random URL):** `start-online-quick.bat`
- **Online (Fixed URL):** `start-online-named.bat`
